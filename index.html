Paragraph Tag

The <p> tag defines a paragraph in HTML. It creates a block of text with margin.The

tag in HTML is used to define a paragraph of text. It automatically adds space above and below the content, making it easier to structure readable blocks of information on a webpage. Paragraphs are essential for organizing text content clearly and semantically. You can apply styles like font size, color, alignment, and spacing using CSS. Though the tag itself doesn’t have required attributes, it supports global ones like id, class, and style. It’s a block-level element, meaning it takes the full width available. Proper use improves both visual presentation and accessibility.
