<header> Tag

The <header> tag represents introductory content, typically a logo, heading, or navigation.

<nav> Tag

The <nav> tag defines a set of navigation links used to move across the website.

<main> Tag

The <main> tag identifies the central content of a document. It is unique per page.

<section> Tag

The <section> tag defines sections in a document such as chapters, headers, or footers.

<article> Tag

The <article> tag is used for independent, self-contained content like blogs or news.

<footer> Tag

The <footer> tag defines the bottom of a document, often used for contact info and links.
